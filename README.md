# Array Average & Basic Input Processing in JavaScript

## Overview

This project demonstrates a simple algorithm for calculating the average of multiple values entered by the user. It focuses on the core concepts of array traversal, summation, arithmetic calculations, and reading input using Node.js.

Although the implementation is simple, it represents one of the most common programming patterns used in competitive programming and coding interviews.

## Concepts Covered

- Arrays
- Loops
- Accumulator Pattern
- Average Calculation
- Mathematical Operations
- Node.js Standard Input
- Time Complexity Analysis

## Algorithm

1. Read the total number of values.
2. Store the input values.
3. Calculate their sum.
4. Divide the total sum by the number of elements.
5. Print the average.

## Complexity

- **Time Complexity:** O(n)
- **Space Complexity:** O(1)

## Technologies

- JavaScript (ES6)
- Node.js
